# Sistema de Gestão Hospitalar VidaPlena – Scripts SQL

Este repositório contém os scripts SQL desenvolvidos para o Trabalho 4 da disciplina de Banco de Dados, com base no mini-mundo do Sistema de Gestão Hospitalar VidaPlena.

## Tecnologias Utilizadas
- MySQL
- MySQL Workbench
- Linguagem SQL (DML)

## Arquivos do Repositório

- `01_inserts.sql`  
  Script responsável pela inserção de dados nas tabelas principais do sistema.

- `02_selects.sql`  
  Script contendo consultas SQL utilizando SELECT, WHERE, ORDER BY, JOIN e GROUP BY.

- `03_updates_deletes.sql`  
  Script com comandos de UPDATE e DELETE, respeitando as chaves primárias, estrangeiras e a integridade referencial do banco.

## Como Executar

1. Criar o banco de dados e as tabelas conforme o modelo lógico do projeto.
2. Executar o arquivo `01_inserts.sql` para povoar as tabelas.
3. Executar o arquivo `02_selects.sql` para testar as consultas.
4. Executar o arquivo `03_updates_deletes.sql` para testar alterações e exclusões de registros.
